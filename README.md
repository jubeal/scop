# Scop, small graphic project

This is a small graphic project from 42 school. The goal was to display a .obj file in a window created with GLFW and using the graphic library OpenGL.

## Compute project

Just run the command
```sh
make
```

## Run project

Execute the computed binary
```sh
./scop
```

### Inputs references

Here is the list of commands you can use in the program:
- `esc`: close window

- `0`: stop the object's rotation
- `1`: slow object's rotation
- `2`: normal object's rotation
- `3`: fast object's rotation
- `I`: change object's rotation

- `arrow left`: move object left
- `arrow right`: move object right
- `arrow up`: move object up
- `arrow down`: move object down
- `page up`: move object away
- `page down`: move object closer

- `O`: change object
- `,`: reduce texture visibility
- `.`: augment texture visibility

- `F`: toggle free cam
- `W`: move front
- `A`: move left
- `S`: move back
- `D`: move right
